Quantum Cricket (Run Chase)

A standalone Vite + React arcade cricket game that teaches quantum mechanics concepts as you score. 

- Pause with P to open the Knowledge Book; flip with ←/→; resume with R
- Aim with ← ↑ →; Swing with SPACE
- Only quantum-themed content is included

Getting started

1) npm install
2) npm run dev

This repository intentionally excludes large PDFs and generated JSON used during local development.
